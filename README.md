# 개요

파이썬 코드를 연습한 내용을 정리한 프로젝트
특정 프로그램에 대한 프로젝트는 아니며, 단순히 코드를 연습하거나 실험한 기록
참고 서적 : Do it! 자료구조와 함께 배우는 알고리즘 입문 파이썬편

## 내용 설명

파이썬 기본 문법(자료형, 조건문, 반복, 함수, 클래스) 등의 기본적인 사용법과 일반적인 알고리즘에 대한 내용 및 구현연습
